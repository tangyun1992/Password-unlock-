（1）利用canvas实现手机密码解锁的UI组件，完成的功能如下：
除了完成基本工作之外，还有如下特色之处：a.同一个数字可重复设置为密码值，同时不限定用户设置的密码长度；
	                                b.利用弧线，可以跨格设置密码，如可以利用弧线设置1,3为密码元素；
	                                c.设置密码的过程中，可以返回上上个密码值，从而取消上一个密码值；
（2）关键技术：
a.利用canvas实现解锁密码面板，包括画圆，画折现等；
b.利用touchStart,touch,touched事件完成对画板的监听；
c.密码匹配的实现：将两次密码利用字符串比较的方法完成；
d.对触屏点的位置与密码面板上每个值的距离进行比较，在设定的半径之内，判定为此值；
e.设置flag标记，显示密码设置和密码验证阶段。
